## Data Sharing Notes

Before publishing any simulation output, confirm that:

1. The file is approved for public sharing.
2. Sensitive project identifiers are removed.
3. The file size is practical for GitHub cloning.

## Recommended Practice

- Keep only one small representative sample file in the repository.
- Keep larger/raw outputs out of the repo.
